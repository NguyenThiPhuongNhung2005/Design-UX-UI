# Lab2 — PetCare (Bootstrap)

Website demo về **petcare** theo phong cách template dạng block (gợi nhớ Mobirise), dùng **Bootstrap 5** + CSS tuỳ biến.

## Cách chạy
- Mở file `index.html` bằng trình duyệt.
- Hoặc dùng VS Code Live Server (nếu bạn có cài extension) để auto reload.

## Cấu trúc
- `index.html`: landing page (hero, dịch vụ, vì sao chọn, bảng giá, review, đặt lịch, FAQ)
- `services.html`: trang dịch vụ chi tiết
- `about.html`: trang giới thiệu
- `contact.html`: trang liên hệ
- `assets/css/styles.css`: giao diện (màu, block, card, responsive)
- `assets/js/main.js`: nhỏ gọn (year, smooth scroll, validate form demo)

## Gợi ý Mobirise (nếu cần đúng yêu cầu bài)
Mobirise là công cụ kéo-thả. Trong phạm vi repo này mình không thể xuất file trực tiếp từ Mobirise, nhưng bạn có thể:
1) Chọn template dạng **Services/Agency** trong Mobirise
2) Thay block hero/services/pricing/testimonials/contact
3) Copy nội dung/structure từ các trang HTML trong repo để khớp chủ đề PetCare

> Nếu bạn gửi tên cửa hàng, địa chỉ, hotline thật, mình có thể thay vào toàn bộ site.
